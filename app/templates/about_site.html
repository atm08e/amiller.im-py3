About this website

Motivation

There are three primary motivations for the creation of this website. Firstly, I wanted a way to keep my full stack engineering skills sharp. This includes the entire stack from the kernel all the way to javascript. It also includes the software engineering process of coming up with requirements, a spec, prioritizing features, to completion and delivery.

Second, I wanted somewhere besides facebook and instagram where I could show off all my trip pictures and express more technological oriented opinions.

Third and most important, I was looking for a fun way to exercise my talent and love for computers! Rule one for me when building this website is that I want to have as much fun with it as possible. Im not going to force any engineering decisions that make working on this website difficult so it becomes "work", eg. no javascript or single page application anymore.

Technologies

  • Python3.5 - Namely the aiohttp framework which leverages the event loop and async/await sytnax.
  • Anisble - For configuration mangement and to deploy amiller-py3 code to "Production".
  • Gitlab - Mostly for their CI/CD integration that is simple enough for my needs and of course source control.
  • Docker - Used mainly in gitlab CI/CD integration for running unit test and to deploy

Code is available at: amiller.im-py3.
