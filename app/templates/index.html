Welcome to Amiller.im

Personal website of Drew, w00t

Welcome to a website full of awesome snowboarding and boating pictures!

7/14/18 - Lots of updates! Switched to concourse from gitlabci. Concourse is just too incredible and useful for other things not to be the one stop shop for CI/CD. The plan is to add linting, packaging, versioning, and green/blueing of the application in the future.

For a short month the website was actually being hosted on a google run kubernetes cluster, but that got really expensive ($50+/month) so I switched the site back to a digital ocean droplet. It was running as Docker image and had true HAnes.

All galleries are alnow powered by modern javascript. Unitegallery.js is to thank for this sweetness. Went through most of 2018 snowboarding photos, hoping to post several more galleries soon.

9/23/17 - Added Key July 2017 gallery!

6/21/17 - Added GFT Part II fishing gallery and links page. All galleries are now using cloud front instead of S3 to host images, noticeable load time improvement.

5/10/17 - Adding some boating galleries from last year! If you have any pictures to add to the galleries please email them or mms them to me. Thank You!

5/09/17 - Refactored the entire code base to be more pythonic, maintainable, and easier to develop for. I feel great about the core code base of the site now and am looking forward to building on top of it for years :). Under the hood many things were improved. The application server is being run with gunicorn instead of just a straight python command. The application also creates a session but it's not being used for anything yet. For development an auto reloading server provided by aiohttp-devtools is being used so that code changes are immediately displayed in the browser. The site also has a new favicon and some content on the about pages has been updated. There is also some other "special" temporary hidden content spread throughout the site.

Next I am thinking of getting some fishing pictures uploaded and then all of the galleries will by uploaded from the original thought of building this website. After that I want to write my first blog post so I have motivation to build out the blog section of the website. I'm feeling a nodejs hating blog is in order :). There are also some css/html tweaks to make that are small but will make this website look more professional and less amateurish. Stay tuned!

5/07/17 - Snowboarding galleries for 2016 & 2017 online! The galleries could use some dynamic formatting but I'm not going to prioritize that right now.

5/06/17 - Iteration 1 of CI/CD completed via gitlab. See the code for it here: amiller.im-py3. Code will be mirrored to Github on a daily basis, but wont serve as the primary repo. Next objective is to get some snowboarding galleries online!

4/29/17 - Redid the website in a full server side rendered python 3 stack. Dedicated post to come soon with my motivations for this, hopefully soon.. Hoping that this website will be funner to work on now that I convinced myself to just stay away from javascript world.

3/02/16 - Website Online! This website is a work in progress, links will be broken, wild cats will apear, and will be updated rapidly in the coming days/weeks
